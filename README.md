# Sujal's Portfolio

## 🌐 Live Website

🔗 **[Open Portfolio](https://chokkarapuwar-sujal.github.io/Portfolio/)**

This is my personal portfolio and digital resume, built to showcase my projects, skills, and experience. The UI is designed to be interactive, fully responsive, and features smooth typing animations.

## ✨ Features

- **Interactive UI**: Engaging user experience with smooth transitions.
- **Typing Animations**: Dynamic text animations using `react-type-animation`.
- **Responsive Design**: Looks great on both desktop and mobile devices.
- **Digital Resume**: A comprehensive overview of my professional experience and skills.

## 🛠️ Tech Stack

- **Framework**: React 19
- **Bundler**: Vite
- **Icons**: React Icons

## 📁 Project Structure

```text
portfolio/
├── public/                  # Static assets like favicon and images
├── src/
│   ├── components/          # Reusable UI components (Navbar, Hero, About, etc.)
│   ├── context/             # React context for global state management
│   ├── data/                # Static data like resume details, projects, and skills
│   ├── styles/              # CSS styling files
│   ├── App.jsx              # Main application component routing and layout
│   └── main.jsx             # React application entry point
├── .gitignore               # Git ignore rules
├── index.html               # Main HTML template
├── package.json             # Project metadata and dependencies
├── README.md                # Project documentation
└── vite.config.js           # Vite bundler configuration
```
